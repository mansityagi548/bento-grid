# Bento Grid 🍱 : 

A **modern, fully responsive Bento Grid** layout built with CSS Grid. This project showcases advanced CSS Grid techniques, responsive design, and clean code structure — perfect for demonstrating layout mastery.


## ✨ Features : 

- **Advanced CSS Grid Layout** using `grid-template-areas`
  
- **Fully Responsive** – Seamless experience from mobile to desktop
  
- **Modern Bento Design** – Clean cards with balanced visual hierarchy
  
- **Hover Animations & Micro-interactions**
  
- **Pixel-perfect** implementation matching the original design
  
- **Mobile-first** approach with smooth breakpoints
